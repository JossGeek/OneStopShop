# OneStopShop
The Sunrise's One Stop Shop application

___

## Frontend
```bash
cd frontend
```
### Installation
```bash
npm install
```

### Execution
```bash
npm run dev
```
___

## Backend
from root
```bash
cd backend
```
### Installation
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows : .venv\Scripts\activate
pip install -r requirements.txt
```

### Execution
```bash
source .venv/bin/activate  # On Windows : .venv\Scripts\activate

```
