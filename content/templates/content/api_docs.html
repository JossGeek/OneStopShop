SUNRISE OSS API Quick Docs

Read-only endpoints for verifying seeded data and browsing offers.

Endpoint Method Description Example
/api/health GET Health check. /api/health
/api/lookups/offer-types GET OfferType lookup values. /api/lookups/offer-types
/api/lookups/domains GET Domain lookup values. /api/lookups/domains
/api/offers GET Offer list with optional filters. /api/offers?limit=10&status=published
/api/offers/{offer_id} GET Single offer detail by UUID. /api/offers/8b2e9a5a-0000-0000-0000-000000000000
curl http://localhost:8000/api/health curl http://localhost:8000/api/lookups/offer-types curl "http://localhost:8000/api/offers?limit=5"
